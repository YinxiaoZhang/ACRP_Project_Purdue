# ACRP_Project_Purdue
The repository contains all the coding files that are used for the ACRP Airport Challenge Project in 2026

The Path_Generation_Dijkstra.py file contains the core path generation algorithm using Dijkstra method. The simulation core code is parking_efficiency_simulation.py, and parking_efficiency_simulation notebook may used to change any necessary simulation parameters with one click to perform the simulation.
